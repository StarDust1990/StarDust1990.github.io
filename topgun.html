Data Collection (Prepare)

On January 6th 2023 I scraped 250,000 tweets using the snscrape Python library, this library uses Twitter's API. The tweet mining date range fell between May 27, 2022 (theatrical release date) to January 6, 2023. Data Assessment and Cleaning (Process)

Mining 250,000 tweets resulted in data that was in desperate need of cleaning. Inspecting and removing rows that contained missing or invalid data was crucial before starting my analysis. A small amount of duplicate values were present in a few columns that should've only possessed unique values, the offending duplicates were then promptly removed. The usernames column had to be cleaned, as some tweets were included solely because the usernames contained the scraping keywords. Upon further inspection it was clear that the locations column was going to be the biggest hurdle. This is because Twitter allows users to have any value for their location, which includes no location. Even though there are many users that use their phones GPS to automatically fill in their accurate location, there was a large amount of data in the location column that was incorrectly formatted, missing or incomprehensible.

Part of my process for cleaning the data in the location column was using a geocoding package in Python. I specifically used the Nominatim geolocation service, through the GeoPy package. Using Python I was able to identify locations in mass quantity and extract additional location info based on the information that was provided. The image below is an example of the output from the Geocode script.

As you can see in the above example, the "Original_Locations" column listed locations that are not real locations such as; "Anywhere with a screen" or "any pronouns" resulted in NaN values (missing) in the Geocoded_Locations, Latitude and Longitude column. This was monumental in distinguishing real locations from invalid location values because not removing these invalid locations would impact the veracity of the data.

Data Preprocessing

Before I am able to process the data through a NLP (Natural Language Preprocessing) model I first must prepare the data to be processed. Normalization, Tokenization and Enrichment are crucial preprocessing steps prior to passing data through an NLP model. Luckily, through the Python packages Transformers and PyTorch, I am able to preprocess the data quickly and effectively to allow us to start processing our data through the Hugging Face roBERTa model.

Sentiment Analysis (Analyze)

Finally, I can process the data through the NLP model to recieve a computed sentiment score. The model that I used was the multilingual XLM-roBERTa-base model from Hugging Face. This model was trained on ~198 million tweets and is finetuned for sentiment analysis. Using the multilingual version of the model was crucial as there was a portion of the data that was not in english. The image below is a portion of my Python script which is taking each tweet and passing it through the model to compute a sentiment score of 1 to 3, and then applying it to a "Sentiment_Score" column. The sentiment score legend is as follows, 1 for negative, 2 for neutral and 3 for positive.

Questions Answered (Act)

What is the sentiment of the average viewer in each Country?

The above dashboard gives you an average sentiment score from any country, for example if we were to pick South Korea, we would see that their Average Sentiment Score is 2.334. Using the Sentiment Legend (1 Negative, 2 Neutral, 3 Positive), we can deduct that the average viewer's sentiment in South Korea is fairly neutral, but leaning more towards positive than negative.

What is the general twitter consensus on Top Gun: Maverick?

From my analysis, 50.2% of each tweet about Top Gun: Maverick was positive, 39.2% was neutral and 10.5% were negative.

Which Country has the highest/lowest average likes?

The Country with the highest average likes is Belize with an average like count of 307.33. The Country with the lowest average likes is Turkmenistan with an average like count of 0.25.

Which Country has tweeted the most about Top Gun: Maverick?

The Country that has tweeted the most about Top Gun: Maverick is the United States.
